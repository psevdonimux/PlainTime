# PlainTime - Планировщик задач

Минималистичный планировщик задач на неделю с приоритизацией по матрице Эйзенхауэра.

## Возможности

- **4 категории задач** — основные, второстепенные, мелкие и ненужные
- **Планирование по дням** — с понедельника по воскресенье
- **Перенос** — перетаскивание задач между категориями
- **Массовый перенос** — копирование задач сразу на несколько дней
- **Импорт/экспорт** — сохранение и загрузка данных в JSON
- **Работает офлайн** — все данные хранятся в браузере

## Хранение данных

Данные сохраняются в `localStorage` браузера. Для переноса на другое устройство используйте кнопки «Экспортировать» и «Загрузить».
