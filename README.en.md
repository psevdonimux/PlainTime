# PlainTime - Task Planner

Minimalist weekly task planner with Eisenhower matrix prioritization.

## Features

- **4 task categories** — main, secondary, minor and unnecessary
- **Day planning** — from Monday to Sunday
- **Transfer** — drag and drop tasks between categories
- **Bulk transfer** — copy tasks to multiple days at once
- **Import/export** — save and load data in JSON
- **Works offline** — all data stored in browser

## Data Storage

Data is saved in browser's `localStorage`. Use "Export" and "Load" buttons to transfer to another device.

## Technologies

- HTML5
- CSS3
- JavaScript
